# Gnostyx

**AI Alert Triage Co-Pilot - Cut Alert Noise. Catch Real Threats.**

Gnostyx is a lightweight AI layer that plugs into your existing SIEM or IDS and automatically filters, scores, and summarizes alerts. No more alert fatigue. No more endless queues. Just the signals that matter.

## Features

- **Automated Triage**: Filters out 50-80% of noise.
- **Smart Context**: Enriches alerts with threat intel and historical data.
- **Instant Summaries**: AI-generated explanations for every alert.
- **Easy Integration**: Plugs into Sumo Logic, Wazuh, and more.
